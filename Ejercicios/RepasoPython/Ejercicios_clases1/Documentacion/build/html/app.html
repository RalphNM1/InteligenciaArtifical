Ejercicios Clases 1

Gestión de Trabajadores de un Hospital

Este módulo proporciona funcionalidades para la gestión de trabajadores en un hospital, incluyendo médicos y enfermeras.

Clases Base

Las clases base utilizadas en este programa se encuentran en el módulo Modelo. - Medico - Enfermera - Trabajador

Funciones Principales

Ejercicios_clases1.cargar_datos_csv(nombre_archivo)[fuente]

Carga datos de trabajadores desde un archivo CSV.

Parámetros:

nombre_archivo – Nombre del archivo desde el que se cargarán los datos.

Ejercicios_clases1.eliminar_trabajador(lista_trabajadores, nif)[fuente]

Elimina un trabajador de la lista dado su NIF.

Parámetros:

  • lista_trabajadores – Lista actual de trabajadores.

  • nif – NIF del trabajador a eliminar.

Ejercicios_clases1.guardar_datos_csv(nombre_archivo)[fuente]

Guarda los datos de los trabajadores en un archivo CSV.

Parámetros:

nombre_archivo – Nombre del archivo donde se guardarán los datos.

Ejercicios_clases1.introducir_datos_trabajador()[fuente]

Solicita y valida la información básica de un trabajador.

Devuelve:

Tupla con los datos del trabajador: NIF, nombre, fecha de nacimiento, número de colegiado (opcional) y sexo.
